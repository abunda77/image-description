# Product Overview

**Reference Describer** is a React web application that generates strict, neutral image descriptions using Google Gemini API. The app is specifically designed to create reference descriptions for textures, poses, and lighting while avoiding facial feature descriptions.

## Core Purpose
- Generate visual descriptions that exclude facial features, hair, ethnicity, and age
- Focus on body posture, clothing details, accessories, actions, objects, environment, and lighting
- Provide consistent output format: `[subject] "(image reference)" [detailed description]`

## Key Features
- Drag-and-drop image upload
- AI-powered description generation via Gemini 2.5 Flash
- One-click clipboard copying
- Responsive design with modern UI
- Strict constraint enforcement for reference use cases

## Target Use Case
Optimized for creating reference material descriptions where facial characteristics should be omitted, making it suitable for artistic reference, texture studies, and pose documentation.
